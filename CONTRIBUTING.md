# Contributing to Neurcode Open Source

Thank you for your interest in contributing! Please see the main [Neurcode repository](https://github.com/sujit-jaunjal/neurcode) for contribution guidelines.

## Quick Start

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Add tests if applicable
5. Submit a pull request

## Package-Specific Guidelines

Each package may have its own contribution guidelines. Please check the README in each package directory.

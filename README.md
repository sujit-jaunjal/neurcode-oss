# Neurcode Open Source

Open-source packages for Neurcode code governance platform.

## Packages

This repository contains the following open-source packages:

- **`cli/`** - Command-line tool for analyzing code changes
- **`diff-parser/`** - Robust parser for unified diff format
- **`github-action/`** - GitHub Action for automated PR analysis
- **`vscode-ext/`** - VS Code extension (coming soon)

- [Main Neurcode Repository](https://github.com/sujit-jaunjal/neurcode) - Main project documentation
